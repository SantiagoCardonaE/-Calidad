Falta conectar la base de datos
Esta app necesita un proyecto de Supabase para guardar la información de forma permanente y compartida entre dispositivos.
  1. Crea un proyecto gratis en supabase.com
  2. Abre SQL Editor → pega el contenido de schema.sql → Run
  3. Ve a Project Settings → API y copia Project URL y anon public key
  4. Pégalos al inicio del archivo HTML, en el bloque SUPABASE CONFIG (busca SUPABASE_URL)
  5. Guarda el archivo y vuelve a abrirlo
